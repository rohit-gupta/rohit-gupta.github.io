Installing Theano with CUDA on Linux Mint 17.2

Published: Fri 31 July 2015
By Rohit Gupta

In Thesis.

tags: Deep Learning Tutorial

Install Theano dependencies from the Ubuntu repos

sudo apt-get install python-numpy python-scipy python-dev python-pip python-nose g++ libopenblas-dev git

Install Theano via pip

sudo pip install Theano

(Optional) Test if Theano is working by running the following command:

python `python -c "import os, theano; print os.path.dirname(theano.__file__)"`/misc/check_blas.py

Note the run time, which cab be found out using $(!!) | grep -i "Total execution time:"

Install the latest Nvidia drivers for your GPU from Menu > Driver Manager

Install CUDA

  • Download repository meta-data deb package from Nvidia: http://developer.download.nvidia.com/compute/cuda/7_0/Prod/local_installers/rpmdeb/cuda-repo-ubuntu1404-7-0-local_7.0-28_amd64.deb

  • Install the repository meta-data:

sudo dpkg -i cuda-repo-ubuntu1404-7-0-local_7.0-28_amd64.deb

  • Update the Apt repository cache

sudo apt-get update

  • Install CUDA

sudo apt-get install cuda

Add the CUDA libraries to your environment:

export PATH=/usr/local/cuda-7.0/bin:$PATH

export LD_LIBRARY_PATH=/usr/local/cuda-7.0/lib64:$LD_LIBRARY_PATH

(Optional) Run the test in Step #3 again to verify that Theano is indeed able to use the GPU (a speed up of 10x or more is likely):

python `python -c "import os, theano; print os.path.dirname(theano.__file__)"`/misc/check_blas.py | grep -i "Total execution time:"
